# Interactive Transformer right in your browser
- Link: https://g4nesh.github.io/interactive-transformer/
- Model Used: DistilGPT2 by Xenova (https://huggingface.co/Xenova/distilgpt2)
- Make sure you close the tab after use as this is high memory. Downloads the model into browser memory each time you open the tab leading to large memory usage (~1-2 GB).
